AmazingNotes!

This application can be used as a todolist and note taking app. You can create/edit/delete notes and also set them as done/undone from a switch, if you are using it as a todolist.

Instructions:

1. To create a new note, go to the notes-tab and click "+" sign in top right and a new view will open. Now write a title and details/description and click save!

2. To view an item, just click it and a new view will open, displaying the items contents.

3. Slide item left to display options, you can either delete the note or edit it's contents.

4. You can also set a note or todo item as checked by clicking the switch!

If you have any quetions or problems, the contact information can be found in contacts-tab
